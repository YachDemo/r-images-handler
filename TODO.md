## 技术栈

| 类别 | 技术选型 |
|------|---------|
| 框架 | Tauri 2.0 |
| 前端 | React 18 + TypeScript |
| 样式 | TailwindCSS + shadcn/ui |
| 状态管理 | Zustand |
| 图片处理 | image + imageproc |
| 并行处理 | rayon |
| 文件监控 | notify |
