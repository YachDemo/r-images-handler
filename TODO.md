# R-Image Studio 路线图与待办事项

## 🟢 阶段一：基础与交互完善 (Workflow & UX) - 优先级：高
- [x] 初始化 Tauri 2.0 + React 19 + TailwindCSS 项目
- [x] 核心后端模块 (文件遍历、图片处理引擎、缩略图系统)
- [x] 基础 UI 框架 (三栏式布局、侧边栏文件夹树、网格/列表视图)
- [x] 批量处理基础 (转换、重命名、缩放、拼图)
- [x] **拖拽支持 (Drag & Drop)**：支持从系统外部拖入图片，自动识别文件夹或文件所在目录
- [x] **全局快捷键系统**：已实现全选 (Ctrl+A)、复制路径 (Ctrl+C)、删除 (Delete)、预览 (Space)、编辑 (Enter)
- [x] **任务队列管理器 (Task Queue UI)**：支持多任务并行处理，实时显示每个任务的进度、状态及错误信息
- [x] **增强版智能拼图**：支持在拼图上添加文本/艺术字，支持图片遮罩（Masking）与自定义形状裁剪
- [x] **搜索与筛选增强**：基于文件名、日期、图片尺寸、格式的实时过滤

## 🔵 阶段二：专业编辑与管理 (Pro Features) - 优先级：中
- [x] 基础编辑器 (RGB、亮度、对比度调节)
- [x] 图片属性与 EXIF 信息展示
- [x] **实时直方图 (Histogram)**：在编辑器侧边栏显示 RGB/亮度分布
- [x] **变量批量水印**：支持在水印中插入动态变量 (如 `{date}`, `{camera_model}`, `{resolution}`)
- [ ] **非破坏性编辑**：引入 Sidecar 文件或本地数据库记录修改序列，支持随时撤销/重做
- [ ] **RAW 格式支持**：集成 `libraw-rs` 或 `rawloader` 支持专业相机原始格式
- [ ] **地理位置地图视图**：利用图片 EXIF 中的 GPS 信息在地图上可视化分布

## 🟣 阶段三：智能化演进 (AI & ML) - 优先级：中/低
- [ ] **本地 AI 超分辨率 (Upscaling)**：集成 ESRGAN 模型，实现图片 2x/4x 无损放大
- [ ] **智能去背景 (Background Removal)**：一键抠图功能 (本地运行模型)
- [ ] **自动标签与分类**：基于内容识别自动为图片添加标签 (如“风景”、“人物”)
- [ ] **相似图片检索**：利用 Perceptual Hashing 算法查找重复或高度相似的照片

## ⚪️ 阶段四：性能与底层优化
- [ ] **GPU 加速预览**：利用 WGPU 或 Shaders 实现编辑器滤镜的实时渲染
- [ ] **智能预加载缓存**：根据用户滚动行为，提前解码下一屏图片的缩略图
- [ ] **内存占用优化**：针对海量图片文件夹的内存占用专项调优
- [ ] **跨平台自动更新**：配置 Tauri 的 updater 流程
